# MCBE-FastBuilder (NormaConstructor)
![CI](https://github.com/NorthernOceanS/NormaConstructor/workflows/CI/badge.svg)

NormaConstructor是一个运行于Minecraft基岩版的快速建造插件，目前单机版已支持Win10版，联机版已支持原版BDS服务端、LXL服务端和BDSX服务端.

如果你正在寻找适合Minecraft基岩版的“小木斧”，那么恭喜你来对地方了！

#### 单机版安装方法：

前往[Releases](https://github.com/NorthernOceanS/NormaConstructor/releases)下载最新版本的Release.mcaddon文件；

双击打开；

在一个世界里打开**全部**的实验玩法后启用刚刚安装好的资源包，启用的方法和启用普通的材质包（如faithful）无异。

#### 更具体的使用方法，请参见 <http://callstackexceed.com/_book/zh-Hans/tutorial.html>

#### 已有的功能：

点击两点生成立方体；生成直线；生成多边形；生成地铁；生成马路；自动生成建筑；生成村庄……

如果想要更多的功能尽管在issues里提
